MagCAMB

By A. Zucca, Y. Li and L. Pogosian

Version 0.0.

Patch for CAMB that computes CMB anisotropies sourced by Primordial Magnetic Fields (PMF)

  • based on CAMB 2015
  • integrated with CosmoMC
  • parts of the code based on the version of "Magnetic CAMB" written by Richard Shaw, provided via private communication

INSTALL:

  • Copy-paste the files in your CAMB directory
  • make

RUN:

  • modify params_mag.ini according to what you want to calculate
  • ./camb params_mag.ini

If you use this code for scientific work, please cite our paper, arXiv:XXXX.XXXXX, Shaw and Lewis paper, and the original CAMB paper.

If you have any question, or find a bug in the code, please contact me at azucca@sfu.ca .
